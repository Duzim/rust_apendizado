## Sobre o Projeto

Este projeto é um repositório de aprendizado e serve como uma documentação pessoal e prática sobre o ecossistema e a linguagem **`Rust`**.

Toda a trilha de estudos e a estrutura das pastas seguem a documentação oficial da linguagem.

> **Referência Oficial:** [The Rust Programming Language (The Book)](https://doc.rust-lang.org/book/title-page.html)
