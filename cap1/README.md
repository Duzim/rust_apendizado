## Primeiros passos do Cargo

- `cargo new`: serve para criar um novo projeto
- `cargo build`: Compila o projeto, criando o executavel com dependencias
- `cargo run`: Compila e roda automaticamente o projeto
- `cargo check`: Verifica se há erros de compilação no projeto
